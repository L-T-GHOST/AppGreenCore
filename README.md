# 🖥️ Data Center 3D Simulator

A **dark mode Flutter Desktop application** that simulates a data center server room with a fully interactive **3D isometric visualization** and live heat/cooling simulation logic.

---
# trigger build
## 📸 App Overview

| Page | Description |
|------|-------------|
| **Dashboard (Page 1)** | Input sliders for CPU, network, temperature, power load & cooling params. Live calculated metrics for estimated room temperature and heat output. |
| **3D Simulation (Page 2)** | Interactive isometric 3D server room with 6 server racks + cooling unit. Animated fan, heat particles, airflow particles, hover tooltips. |

---

## ✅ Prerequisites

Make sure you have:

- **Flutter SDK** ≥ 3.3.0 installed and on your PATH
- **Windows Desktop** target enabled

To enable Windows desktop if not already done:
```bash
flutter config --enable-windows-desktop
```

Verify your setup:
```bash
flutter doctor
```

## 🧮 Simulation Logic

```
ServerHeat     = CPUUtilization × 0.6 + NetworkTraffic × 4.0
RoomTemp       = 18°C + (ServerHeat / (CoolingEfficiency × FanSpeed/100 × 12))
HeatOutput(kW) = ServerHeat × PowerLoad / 100 × 0.05
```

### Server colour states:
| Temperature | Colour | Status |
|-------------|--------|--------|
| < 60°C | 🟢 Green | Normal |
| 60–75°C | 🟡 Yellow | Warm |
| > 75°C | 🔴 Red | Overheating |

---

## 🎨 Visual Features

- **Isometric 3D renderer** built with Flutter `CustomPainter` — no external 3D engine
- **Depth sorting** (painter's algorithm) for correct occlusion
- **Per-server glow** based on thermal load
- **Animated cooling fan** — speed scales with fan speed slider
- **Heat particles** — orange particles rise from hot servers
- **Airflow particles** — cyan particles flow from cooling unit
- **LED indicators** on server rack front panels
- **Vent lines** on cooling unit face
- **60 fps animation** via `AnimationController`

## 🛠️ Troubleshooting

**"No Windows desktop device found"**
```bash
flutter config --enable-windows-desktop
flutter doctor
```

**Build errors with provider**
```bash
flutter pub cache repair
flutter pub get
```

**Low frame rate**
The 3D simulation uses CPU-side painting. Make sure you're running a `Release` build for best performance:
```bash
flutter run -d windows --release
```
